# Ex. No. 5 - SIMULATION-OF-CARTESIAN-3D-PRINTER-MACHINE

## DATE: 13/03/2026
## AIM:
### To simulate the construction of cartesian 3D printer and to get in-depth knowledge of mechatronics of cartesian 3D printers.

![image](https://github.com/Sellakumar1987/Ex.-No.-3---SIMULATION-OF-CARTESIAN-3D-PRINTER-MACHINE/assets/113594316/69572917-1257-45d7-bf57-ff48a6e5a711)

## REQUIREMENTS:
### ●	System - Windows 7 or higher, 1 GB RAM.

## PROCEDURE:
## X-Axis Assembly
### 1. Choose **X-Axis Assembly** from the list of available options.
### 2. The components related to the X-axis will be displayed on the screen.
### 3. Select the parts in the order shown.
### 4. When the first component is selected, it will appear in the blank space on the left side of the screen.
### 5. Each correctly selected component will automatically assemble with the previously selected parts.
### 6. If an incorrect order is chosen, a pop-up message will appear indicating the correct part to be selected.

---

## Y-Axis Assembly
### 1. Select **Y-Axis Assembly** from the available list.
### 2. The parts corresponding to the Y-axis will be displayed on the screen.
### 3. Choose the components following the sequence provided.
### 4. After selecting the first component, it will appear in the blank space on the left side of the screen.
### 5. The correctly selected components will assemble with the previously selected parts.
### 6. If the sequence is incorrect, a pop-up message will appear indicating the correct component to choose.

---

## Z-Axis Assembly
### 1. Select **Z-Axis Assembly** from the visible list.
### 2. All the parts associated with the Z-axis will appear on the screen.
### 3. Choose the components according to the given order.
### 4. When the first component is selected, it will appear in the blank space on the left side.
### 5. Correctly selected components will assemble with the previously selected parts.
### 6. If the wrong sequence is followed, a pop-up notification will appear showing the correct part.

---

## Complete Axis Assembly
### 1. Select **Complete Axis Assembly** from the list of options.
### 2. The components related to the complete axis assembly will be displayed.
### 3. Select the parts in the order provided.
### 4. The first selected component will appear in the blank space on the left side of the screen.
### 5. Each correct component will assemble with the previously selected parts.
### 6. If an incorrect sequence is chosen, a pop-up message will indicate the correct part to be selected.

---

## Axis Movement Demonstration
### 1. Adjust the **X-axis slider** or enter a value within the allowed range and press **Enter** to observe the movement of the X-axis assembly.
### 2. Adjust the **Y-axis slider** or enter a value within the allowed range and press **Enter** to observe the movement of the Y-axis assembly.
### 3. Adjust the **Z-axis slider** or enter a value within the allowed range and press **Enter** to observe the movement of the Z-axis assembly.
